free-gpu | provider workflow planner

Pick the right free or near-free compute path in minutes.

llmfit tells you what fits locally. free-gpu tells you what to do next: stay local, use a free tier, spend a few credits, or move to a grant-style provider.

Quick start

Install once, then choose the surface that matches your workflow.

TUI

Interactive planner

Browse providers, filter options, and compare plans in a local Textual UI.

pip install free-gpu
free-gpu ui
Hosted MCP

Remote HTTP endpoint

Use this when your MCP client supports remote HTTP and you want zero local setup.

https://free-gpu.vercel.app/mcp
Local MCP

Local stdio server

Use this when your coding agent can launch MCP tools directly on your machine.

pip install free-gpu
free-gpu-mcp

CLI examples

Use direct commands when you do not need MCP.

pip install free-gpu
free-gpu providers --workload inference --budget free
free-gpu plan --workload agent-loop --budget under-25 --task-hours 3 --parallel-jobs 4 --requires-api

What it does

Assess

Estimate the compute lane

Figure out whether a task looks like a burst, session, heavy, or grant-scale job.

Rank

Find realistic providers

Rank free tiers, cheap credits, and grant-style programs for the actual workload.

Plan

Build a workflow

Split the work between local and remote stages instead of treating every task the same way.

MCP setup

Hosted MCP

Use this when your client supports remote HTTP MCP and you want the simplest setup. 

https://free-gpu.vercel.app/mcp

Best for Codex, Claude Code, Cursor, or any MCP client that can call a remote endpoint.

Local MCP

Use this when your client launches stdio MCP tools on your machine.

pip install free-gpu
free-gpu-mcp

For Codex, the reliable local registration is codex mcp add free-gpu-local -- free-gpu-mcp.

Important note

GitHub Pages hosts the docs only. The planner itself runs either as a local MCP tool or as the hosted MCP endpoint on Vercel.

Provider lanes

Free tier

Fast and lightweight

Good for quick inference, demos, notebooks, and experiments.

Credits / under-25

Cheap medium runs

Good for longer sessions, eval loops, and moderate fine-tunes.

Grant / program

Heavy tasks

When VRAM and runtime grow, the planner shifts toward allocations, programs, and grants.
